## Setup
- Add `nuxt-express-module` dependency using yarn or npm to your project
- Add `nuxt-express-module` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-express-module',

    // With options
    ['nuxt-express-module', { /* module options */ }],
 ]
}
```
